Divs Coding Challenge

In this lesson you will:
  1. Learn all about Divs
  2. Style with Divs

The <div> element is a divider.

That means that it will always start a new line and will stretch out as much as it can to the left and right.

The <div> is also a container for other HTML elements.

Instead of styling in CSS, you can also style in a div, this is helpful when you want to change just one part of your website.

Let's get started!

Create a new folder in your project folder. Name it Divs.

Create a new html file called index.html.

Create a new css file called style.css.

Link the style sheet in the header of your html file.
