Intro to CSS Grid

In this lesson you will:
  1. Learn to use CSS Grid.
  2. Arrange divs on your page using a grid.
  3. Finally solve the frustrating problem of placing items next to each other on a web page.

What You Will Make

Get Started!

If you are using repl.it, make a new repl.

Head over to CSS-Tricks to see some basic CSS Grid layouts.

The basic idea we need to understand is that a grid is made of rows and columns.
Think of a chess/checkers board. It has 8 rows (horizontal) and 8 columns (vertical).
CSS Grid uses a grid to organize items on a web page.
HTML

Inside your repl, open up the index.html file.

Copy and paste the following code inside the body of your page.
<div class="board">
<div class="square">1</div>
<div class="square">2</div>

Each of these divs will become a box on our monopoly board.

Once we see our webpage, we will replace the numbers with our own property names for our monopoly board.
CSS

Open up style.css.

Add the following code to your css file.
.board {
display: grid;
justify-content: center;
align-content: center;
grid-gap: 1rem;
grid-template-columns: repeat(5, 10vw);
grid-template-rows: repeat(5, 10vw);
}

This code for the board class implements CSS Grid.
It is using 5 rows and 5 columns to make a simple grid.
Which part of the code indicates the number of rows and columns?
  • infoHint
    grid-template-rows and grid-template-columns

Let's style our divs a bit more. Add the following code:
.square {
background: #bfdbae;
text-align: center;
display: flex;
align-items: center;
justify-content: center;
}

This colors our divs, and centers the text in each div.

Now we need to arrange our properties in a square around the edge of the page.

Add this code:
.square:nth-child(1) {
grid-row: 1;
grid-column: 1;
border: black 5px dashed;
}

This code styles the 1st child of square. I.e. the first div with the class square.
That div is put in row 1 and column 1.

Save the code and run to see what it looks like.

The second child of square will be similar, try adding the following code:
.square:nth-child(2) {
grid-row: 1;
grid-column: 2;
}

Finish adding code for each square, indicating which row and column it should be in.

Customizing the Properties

Now let's customize our monopoly properties!

Inside index.html, replace the numbers in each div with the monopoly property name you'd like to use.

Now head back to style.css, and let's add some property colors.

To add a colored bar like at the top of a monopoly property, we will use a border.

Try adding the following code inside one of your squares:
border-bottom: blue 10px solid;

border-bottom adds a border at the bottom of a div.
border-top adds a border at the top.
border-left and border-right add borders on the left and right sides of a div.

Use borders to add colored bars like the monopoly properties to your board.
