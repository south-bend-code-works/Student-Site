Link HTML+CSS Files Together

Add this line of code in between your two <head> </head> tags.

<link rel="stylesheet" href="style.css">

All together, your code should look like this:


<!DOCTYPE html>


<html>


<head>


<title>My Website!</title>>


<link rel="stylesheet" href="style.css">


</head>


<body>


<h1>Here are fun facts about me:</h1>


<h4>I love coding</h4>


<p>I love coding because I can build websites and games, using code!</p>


<h4>My favorite color is blue</h4>


</body>


</html>
