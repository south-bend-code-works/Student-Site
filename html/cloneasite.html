Clone a Site

In this lesson you will:
  1. Clone your favorite site!
What You Will Make
Click here for an example!

Cloning a Site or using code to build a look-a-like of a website that you’re familiar with is a great way to test your coding skills. You won’t pass another website off as your own or anything but it is a fun way to see if you can code something just by looking at it.

Let's get started!

Look up a website that you like!

Right click anywhere on the site, select inspect.

You will now have all of this information:

Click on the arrow at the top, and then when you mouse over parts of the website, it'll show you the code on the page.

After clicking on the arrow

See if you can use the skills that you've learned in HTML and CSS to code a site that looks just like the real one!
Show and instructor your finished website!
Congrats! Project complete!
