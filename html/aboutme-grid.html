About Me with Grid

In this lesson you will:
  1. Learn to use CSS Grid.
  2. Arrange divs on your page using a grid.
  3. Revise your About Me page to use CSS Grid.

What You Will Make

Get Started!

If you are using repl.it, make a new repl.

Head over to CSS-Tricks to see some basic CSS Grid layouts.
In this example, we are using the Flexible 2 Column Layout with Header and Footer.
You can use a different layout if you'd like.

The basic idea we need to understand is that a grid is made of rows and columns.
Think of a chess/checkers board. It has 8 rows (horizontal) and 8 columns (vertical).
CSS Grid uses a grid to organize items on a web page.

Our grid for this project will just have three rows and two columns.
HTML

Inside your repl, open up the index.html file.

Copy and paste the following code inside the body of your page.
<header>
Header
</header>
<main>
Main
</main>
<aside>
Sidebar
</aside>
<footer>
Footer
</footer>

These sections will make up our page.

You will transfer your information from your old About Me page into this one.
CSS

Open up style.css.

Add the following code to your css file.
body {
display: grid;
grid-template-columns: 75% 25%; /* adjust column size here */
grid-template-rows: 10vw 30vw 10vw; /* adjust row height here */
grid-gap: 1em;
font-family: Arial, Helvetica, sans-serif;
}

This code sets up our grid using two columns and three rows.

Now we will make our header and footer extend across both columns.
header,
footer {
grid-column: 1 / span 2;
}
  • infoHint
    grid-column: 1 / span 2; tells the header and footer to start in column 1 and span across two columns.

Now let's add some colors and spacing. Put the following code inside style.css.

header,
main,
aside,
footer {
background: #6babe0;
display: flex;
align-items: center;
justify-content: center;
}

This gives our sections a background color, and sets the alignment and justification of content.

Feel free to adjust the colors, and add any padding or margins to the sections as you see fit.

Save the code and run to see what it looks like.

Now add your content to the page! Start with your name in the header.

Maybe add an image of yourself in the aside.
Style it up!

Dive into the CSS and style your page as you'd like it!

You can use different colors, fonts, etc...

Try adding lists and links inside the index.html file.

Embed some videos into the main section.

Adjust your content and styling until it's just how you want it!
Just for fun!

Try adding another row into your page.
Add another two sections in HTML, and then change the number of rows in CSS.

Try adding a background image to one section.

Congrats! Project complete!
