Collage with CSS Grid

In this lesson you will:
  1. Create an image collage using CSS Grid.
  2. Adjust grid spacing based on viewport size.
  3. Make your site responsive to window size changes.

What You Will Make

Get Started!

If you are using repl.it, make a new repl.

For this project we will be making an image collage.

The basic idea we need to understand is that a grid is made of rows and columns.
Our grid will have 5 columns and 10 rows. You can change these numbers as necessary.

Gather the images you want to put in your collage. For our tree collage, we used 8 images. You can use more or less than that, but use at least 5 images.

On a piece of paper, sketch out the arrangement you'd like your images to be in. Make a plan for which ones will go where, and how big the different images should be.
HTML

Inside your repl, open up the index.html file.

Start by adding a div with the class "container".

Above the container div, add a h1 tag, with the name of your collage inside it.
<h1>Trees</h1>

Now let's add image tags for each of your images.
<img src="SOURCE HERE" class = "images" id="image1" alt="ALT TEXT HERE">
</img>

Add the source, alt text, image class, and a unique id for each image.

Save your work and take a look. All your images will be showing up one after another on the page. Next we will arrange and size them using our grid.
CSS

Open up style.css.

Start with the container selector. Add a container selector, and adjust the following properties:
display:grid;
grid-template-rows: repeat(10, 5vw);
grid-template-columns: repeat(5, 1fr);
grid-gap: 1em;
You can adjust the grid-gap to change the spacing between your images.

Before we make our image grid, let's setup the h1 heading. Using the h1 selector, we will make the font size responsive to the window size. Set the following properties, and use the calculation below for font-size.
h1 {
text-align: center;
font-family: Arial, Helvetica, sans-serif;
font-size: calc(32px + 6 * ((100vw - 320px) / 680));
}
To learn about fluid typography, head over here.

Now let's set up our image class.
Add a selector for .image and use the following properties:
.images {
width: 100%;
height: 100%;
object-fit: cover;
}
Setting the width and height to 100%, and the object-fit to cover allow us to resize the images with the page. Be sure that you don't set the image height and width in HTML.

We're ready to arrange our collage.
Right now your images should all be showing up in a grid, but not sized properly.

Grab your layout sketch again.
Try to draw a grid lightly over your desired arrangement. See example below.

Start with image1, note what rows and columns it spans. Our first tree image spans rows 1 through 4 and column 1 through 4.
#image1{
grid-row: 1 / 4;
grid-column: 1 / 4;
}

Then move on to image2.
#image2{
grid-row: 1 / 5;
grid-column: 4 / 6;
}

Adjust the row and column spacing for each image, one at a time, to match your grid sketch.
If you need to adjust the number of rows and columns in the grid, just go back to the container selector, and adjust the repeat numbers to change how many rows and columns are in the grid.

Work slowly, and test after each change to see how your collage looks.
Make It Pretty

Add a background color! Use the body selector to set a background color.

Try adding a border to your images. Use the border property in the .image class.

Try adjusting the border-radius property to soften the corners of your images.

Add a footer with your name and copyright.

Test out your work! Open the site in a new window, and resize the window to be sure your images and header resize properly.

Congrats! Project complete!
