Hover Animation

In this lesson you will:
  1. Create hover animations for divs, buttons, and any other item on a webpage.
What you will make

An interactive hover feature for websites.

Hover here!
Get Started!

Create a new folder in your project folder. Name it "Hover Animation".

Create a new html file called "index.html".

Create a new css file called "style.css".

Link the style sheet in the header of your html file.
HTML

Set up the doctype, html, head and body tags in your html page.

Add a div for the hover animation.

Give your div a class like "section".
  • infoHint
    <body>
    <div class="section">
    </div>
    </body>

CSS

Create a style for your section class.
  • infoExample
    .section {
    background-color: black;
    color: white;
    padding: 20px;
    border: 2px solid green;
    width: 100%;
    }

Now all we need to do is add another class selector in CSS for .section:hover

This tells the CSS that when the mouse is over this div, to change the style to the .section:hover style.

Try using the same style as the section class, but changing one of the colors.
  • infoExample
    .section:hover {
    background-color: green;
    color: white;
    padding: 20px;
    border: 2px solid green;
    width: 100%;
    }

Save your project and try it out!

Now try changing other parts of the section:hover style.
  • infoExample
    .section:hover {
    background-color: green;
    color: black;
    padding: 20px;
    border: 4px dashed black;
    width: 100%;
    }

You can add a transition time so that the hover effect changes smoothly.

In the original section class selector in CSS, add transition: 2s;

You can use the :hover selector on any element in CSS! You can use it on headings, links, buttons, paragraphs, divs, lists and even more.

Congrats! Project complete!
