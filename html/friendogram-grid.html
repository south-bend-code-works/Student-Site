Friend-o-gram with Grid

In this lesson you will:
  1. Make your own collage of your friends and things you like!
  2. Use CSS Grid to arrange your images.

What You Will Make

Brainstorm

Pick out 6 photos of you and your friends.
Your Coding Environment

Create a new repl.it workspace for HTML, CSS, JS.
Divs

Use the <div> tag to make one div, with the class name "wrapper".
  • help I need a hint
    <div class="wrapper"> </div>

Inside the wrapper div, use the <div> tag to divide your website into six divs.

Our grid is made up of 2 rows and 3 columns.

Give each div a unique id, like "one", "two", "three", etc...
  • help I need a hint
    <div id="one"> </div>
CSS

Head over to the style.css page.

Add the wrapper class to the css file by using .wrapper{ }

Turn on CSS grid by using display: grid;, and set the number of rows and columns using grid-template-rows and grid-template-columns
  • help I need a hint
    .wrapper{
    display: grid;
    grid-template-rows: 30vw 30vw;
    grid-template-columns: repeat(3, 1fr);
    }

Add a grid-gap of your choosing.

Add a css selector for each id.
#one{ }
#two{ }
etc...

Set the row and column value for each id.
grid-column: 1;
grid-row: 1;
Add Photos

To set the background images, use the background-image property and give each div a background image.
  • help I need a hint
  • help I need a hint
    #one{
    grid-row: 1;
    grid-column: 1;
    background-image: url('yourImagePath.png');
    background-size: cover;
    }
Hover Effect

Find the div that you want to add a hover effect to.

Let’s say that you want to add a hover effect to id #box1:

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

Make a Copy of that #id so that there are two in CSS

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

Change the characteristics of 1 of the #ids

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

#box1 {
background-color: red;
color: blue;
padding: 20px;
border: 2px solid green;
width: 80%;}

Add :hover to the #id that you want to change when you hover over the square

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

#box1:hover {
background-color: red;
color: blue;
padding: 20px;
border: 2px solid green;
width: 80%;}

If the change happens too quickly, you can add transition time

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;
transition: 2s;}

#box1:hover {
background-color: red;
color: blue;
padding: 20px;
border: 2px solid green;
width: 80%;}

Congrats! Project complete!
