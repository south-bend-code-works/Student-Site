Meme Divs

In this lesson you will:
  1. Make your own memes using HTML and CSS!

What You Will Make

Brainstorm

Pick out a couple of meme photos you want to use.

Download these photos to your computer and give them descriptive names.
Your Coding Environment

Open repl.it and make a new repl using HTML, CSS and JS.

Upload your meme images into your repl.it workspace.
  • help I need a hint
Divs

On index.html, use the <div> tag to add a space for each meme.

Give each div a class with a name for the meme.

<div class "memeName"> </div>

Between each opening and closing div tag, add the text for your memes.
CSS

Click on the style.css file in repl.it.

Make a class for each meme. use a . to indicate the class.
  • help I need a hint
    .memeName { }

Set the background-image: for each meme div to be the meme images you have chosen.

Each meme div needs a width and height. Set these using height: and width:

Set the background-size: to contain.
  • help I need a hint
    .memeName{ background-size: contain; }

Set the
font-size:
text-align:
padding:
to values of your choosing. Test out different values to see what you like.

The standard meme font is impact, so set the font-family: to impact.

Set the color: to white.
  • help I need a hint
Text Shadow

There are two ways to add a black outline to our white text.

We can use the text-shadow: property.
text-shadow: -1px 0 black, 0 1px black, 1px 0 black, 0 -1px black;

Or we can use the webkit.
-webkit-text-stroke: 1px black;
-webkit-text-fill-color: white;

Pick one of these, and add it to your memeDiv class in CSS.

Now adjust the CSS properties as you like to get the desired effect!

You may need to use the line break <br> tag in your HTML file to separate text if you want a meme with top and bottom text.

Congrats! Project complete!
