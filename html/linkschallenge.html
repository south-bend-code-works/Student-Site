Links Coding Challenge

In this lesson you will:
  1. Learn all about HTML Links
  2. Create dynamic links.

Links are found in nearly all web pages. Links allow users to click their way from page to page.

HTML links are hyperlinks. Hyperlinks are defined with the HTML <a> tag

But, there are different ways to display your links!

Let's get started!

Create a new folder in your project folder. Name it Hyperlink.

Create a new html file called index.html.

Create a new css file called style.css.

Link the style sheet in the header of your html file.

By default, a link will appear like this (in all browsers):
  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red
