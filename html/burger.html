Build a Burger

In this lesson you will:
  1. Make your very own burger designed with just HTML & CSS.

What You Will Make

Get Started!

Create a new folder in your project folder. Name it "Burger".

Create a new html file called "index.html".

Create a new css file called "style.css".

Link the style sheet in the header of your html file.
HTML

Set up the doctype, html, head and body tags in your html page.

We will divide up the burger using divs.

Create a div for each section of your burger. Give each div an appropriately named class.
  • infoHint
    <div class=”topbun”></div>
    <div class=”cheese”></div>
    <div class=”tomato”></div>
    <div class=”pickle”></div>
    <div class=”patty”></div>
    <div class=”patty”></div>
    <div class=”bottombun”></div>

We want to use classes because there can always be more than one class, just like there can be more than one tomato or patty. Id’s are used for only one thing.
CSS

Create a style for each class in your CSS file.
  • infoHint
    .cheese {
    width: 20px;
    height: 100px;
    background-color: yellow;
    }

To round the corners of a div, we will use the border-radius property.

border-radius: 100px 100px 20px 20px;

The first value applies to top-left corner, second value applies to top-right corner, third value applies to bottom-right corner, and fourth value applies to bottom-left corner.

Border-radius defines the radius of the element's corners.

Visit W3Schools to find answers to your CSS questions.

Keep on styling until your burger is complete!

Congrats! Project complete!
