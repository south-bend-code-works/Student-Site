Pattern Animation

In this lesson you will:
  1. Create a pattern animation using keyframes.
What you will make

A fun pattern animation.

Get Started!

Create a new folder in your project folder. Name it "Pattern Animation".

Create a new html file called "index.html".

Create a new css file called "style.css".

Link the style sheet in the header of your html file.
HTML

Set up the doctype, html, head and body tags in your html page.

Add an even number of divs for your patterns.

Give every other div the same class name.
  • infoHint
    <div class="red"></div>
    <div class="blue"></div>
    <div class="red"></div>
    <div class="blue"></div>
    <div class="red"></div>
    <div class="blue"></div>

We label each div with either an id or class. Id's can be used if that div or section on your website is only going to appear once.
Classes can be used if that section of your website will appear multiple times.

CSS

Add width, height, and a solid background-color to your CSS file just so that we can see if everything's working correctly.

To identify your div in CSS you can do a few things:

If every div on your website is the same, write the word div.
  • infoExample
    div {
    background-color: red;
    width 100%;
    height 100px;
    }

If you added an id, write a hashtag before the name of your id.
  • infoExample
    #red {
    background-color: red;
    width:100%;
    height:100px;
    }

If you added a class, write a period before the name of your class.
  • infoExample
    .red {
    background-color: red;
    width:100%;
    height:100px;
    }

Create a style for each of your classes, using different background-colors.
Be sure to save and make sure your divs show up on screen.
Adding Images and Animating

Find some cool pattern gifs on google images.

Add one pattern gif to each class as a background-image.
  • infoExample
    .red{
    background-image: url('red.gif')
    }

Now we will animate each div using keyframes!

Keyframes are CSS rules that tell our animation what to do.

Use “@keyframes” as a new selector followed by the name of your animation and braces
(make sure your @keyframes is outside of the brackets for the element you're animating!)
  • infoExample
    @keyframes background-ScrollRight{

    }

To create the animation, we need to add a "from" and a "to" to our keyframe.

“from” is what you want the element you're animating to look like at the start of your animation.
“to” is what you want it to look like at the end. Each one of them has their own set of braces to put rules into, so you can change more than one thing at once!
  • infoExample
    @keyframes background-ScrollRight{
    from{ }
    to { }
    }

To make our background-image scroll right, we need to tell it to go from a background-position of (0, 0), to a background-position of (200px, 0).
  • infoExample
    @keyframes background-ScrollRight{
    from{background-position: 0, 0; }
    to {background-position: 200px, 0; }
    }

See if you can define a background-ScrollLeft animation using keyframes.

Now all that's left is to add the animation to our class selector in CSS.

Add the line animation: background-ScrollRight 10s linear infinite; to one of your classes.

See if you can add the background-ScrollLeft animation to your other class.
  • infoHint
    .red{
    background-image: url('red.gif')
    width: 100%;
    height: 100px;
    animation: background-ScrollLeft 10s linear infinite;
    }

Congrats! Project complete!
