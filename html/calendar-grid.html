Calendar with CSS Grid

In this lesson you will:
  1. Create a calendar using CSS Grid.
  2. Adjust grid spacing based on viewport size.
  3. Add hover animations to your calendar.

What You Will Make

Get Started!

If you are using repl.it, make a new repl.

For this project we will be making a calendar.

The basic idea we need to understand is that a grid is made of rows and columns.
Our grid will have 7 columns (one for each day), and 5 or 6 rows (based on how many days are in the month).
CSS Grid uses a grid to organize items on a web page.
HTML

Inside your repl, open up the index.html file.

Start by adding a div with the class "calendar".

Inside the calendar div, add a header tag, with the month name inside it, using h1.
<header>
<h1>October 2020</h1>
</header>

Now let's add an unordered list with the class "weekdays".
<ul class = "weekdays">
</ul>

Inside the unordered list, we need to add seven list items, one for each day of the week.
<ul class = "weekdays">
<li>
</li>
.
.
.
</ul>

Inside each list item, we will add an abbreviation tag, one for each day:
<li>
<abbr title="S">Sunday</abbr>
</li>

Now we need another unordered list, this one for our days. Give it a class "day-grid".
<ul class = "day-grid">
</ul>

Inside the day-grid unordered list, we need a list item for every day of the month (31 days in October).
<ul class = "day-grid">
<li>1</li>
<li>2</li>
.
.
.
</ul>

In addition to the days in this month, we need a list item for each day in the previous month that shows up in the first week of this month. We will give these the class "month-prev".
<ul class = "day-grid">
<li class="month-prev">29</li>
<li class="month-prev">30</li>
<li>1</li>
<li>2</li>
.
.
.
</ul>

Finally, we need list items for the days of the next month that show up in the last week of this month. We will give these the class "month-next".
<ul class = "day-grid">
<li class="month-prev">29</li>
<li class="month-prev">30</li>
<li>1</li>
<li>2</li>
.

<li>30</li>
<li>31</li>
<li class="month-next">1</li>
<li class="month-next">2</li>
</ul>
CSS

Open up style.css.

Start with the header selector. Add a header selector, and adjust the following properties:
display: flex;
align-items: center;
justify-content: center;
background: PICK A COLOR;
color: PICK A COLOR;
text-align: center;

Now let's make the font size responsive to the window size. Use the calculation below for font-size.
font-size: calc(16px + 10 * ((100vw - 320px) / 680));
To learn about fluid typography, head over here.

Now let's make our calendar into a grid!
Add a selector for ul and use the following properties:
display: grid;
grid-template-columns: repeat(7, 1fr);
grid-gap: 1em;
padding: 0;

Now add a selector for li, with the following properties:
display: flex;
align-items: center;
justify-content: center;
list-style: none;
margin-left: 0;
font-size: calc(16px + 5 * ((100vw - 300px) / (1600 - 300)));
We are using a similar calculation to make the font size responsive to window size.

Let's add some color!
Add a selector for list items inside unordered lists with the class "day-grid".
  • infoI Need a Hint
    ul.day-grid li{ }

Add these properties:
background-color: #eaeaea;
border: 1px solid #eaeaea;
height: 12vw;
max-height: 125px;

Now we will style the days of the week.
ul.weekdays abbr[title] {
border: none;
font-weight: 800;
text-decoration: none;
}

Save the code and run to see what it looks like.

We want only the days this month to be gray, and the ones from last month and next month to be white, or a different color.
To select them, use the following selectors (update the nth child numbers appropriately to match your calendar):
ul.day-grid li:nth-child(1),
ul.day-grid li:nth-child(2),
ul.day-grid li:nth-child(34) {
background-color: #fff;
}

Now we need to make our site responsive. We will use a media query to specifiy the style when the screen is narrower than 800px.
Add the following code:
@media all and (max-width: 800px) {
ul {
grid-gap: .25em;
}

ul.weekdays li {
font-size: 0;
}

ul.weekdays > li abbr:after {
content: attr(title);
font-size: calc(16px + 10 * ((100vw - 300px) / (1600 - 300)));
text-align: center;
}
}

Try resizing the window, and see how your calendar responds!
Hover Animation

Now let's add a hover animation when the user hovers over a date.

Add one more selector in CSS. This one for hovering over list items in the unordered list with the class day-grid.
  • infoI Need a Hint
    ul.day-grid li:hover

Set a new background-color, a border, a bigger font size (use the calculation from earlier, just increase the base font size from 16 to something like 32).
  • infoI Need a Hint
    background-color: blue;
    border: 1px solid blue;
    font-size: calc(32px + 10 * ((100vw - 300px) / (1600 - 300)));

You can also add transition time to make it smoother.
  • infoI Need a Hint
    transition: 1s;

Test out your work!
