Hover Effect

Find the div that you want to add a hover effect to.

Let’s say that you want to add a hover effect to id #box1:

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

Make a Copy of that #id so that there are two in CSS

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

Change the characteristics of 1 of the #ids

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

#box1 {
background-color: red;
color: blue;
padding: 20px;
border: 2px solid green;
width: 80%;}

Add :hover to the #id that you want to change when you hover over the square

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;}

#box1:hover {
background-color: red;
color: blue;
padding: 20px;
border: 2px solid green;
width: 80%;}

If the change happens too quickly, you can add transition time

#box1 {
background-color: black;
color: white;
padding: 20px;
border: 2px solid green;
width: 100%;
transition: 2s;}

#box1:hover {
background-color: red;
color: blue;
padding: 20px;
border: 2px solid green;
width: 80%;}
