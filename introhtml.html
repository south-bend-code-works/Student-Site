Manipulate Code

In this project, you will manipulate a website's code to change images, background-colors, and text!

What You Will Make

Scroll through the website below for an example of the type of website that you will make.

Step 1: What is HTML?

HTML stands for HyperText Markup Language, and is the language used to make webpages.

Step 2: CodePen

You'll be using a website called CodePen to write HTML. It should look like this

See the Pen SBCS Code by Camisa (@camisa) on CodePen.

The code that you can see on the left in CodePen is HTML.

On the right of CodePen you can see the webpage that the HTML code made.

HTML uses tags to build websites.

The example tag: < p > stands for paragraph. Can you spot more tags in the code above?

< h1 > < / h1 >

< h2 > < / h2 >

< iframe > < / iframe >

< img >

< html > < / html >

< head > < / head >

< body > < / body >

You may have noticed that HTML tags normally come in pairs like < p > and < / p >

The first tag in a pair is the start tag, the second tag is the end tag.

Step 3: Start Coding

Open the starter project !

Note that if you refresh the codepen site, any changes you made will be lost.
HTML

Header tags (ex. h1, h2, h3, etc.) change text.

Header tags look like this:

<h1>

<h2>

Find the <h1> tag that says <h1>Your Name Here<h1> in CodePen.

Replace ONLY the words "Your Name Here" with your first name or nickname.
For example:
<h1> Anna </h1>

Keep going to change the other headers (<h1> and <h2>) and paragraphs (<p>)in the website.

Scroll through your website and double check that you successfully updated all of the text!
CSS

CSS stands for Cascading Stylesheet.

With CSS you can change a website’s background-images, types of fonts, text colors, and background-colors to style the page.

We’ll be using what’s called inline-CSS to style our websites!

First, we're going to change your background picture
Background Images

Go to Google images and search for your favorite image

Once you find the the picture on Google, right click on the image and select “Copy image address” from the drop down menu.

Return to your code and paste the copied image address into your code.

You've now successfully changed the background image!
Favorite Meme

You’ve learned how to update images! Replace the images in this section with pictures of your favorite memes.
Text Colors

The property that changes text-color is color.

For example, color:white; means that your text-color is white.

To change the text-color, switch "white" with the name of another color.

Now you try it!

Visit colors.commutercreative.com for access to more colors:

Change all of the different text colors on your page!
Background Colors

The way that you just changed text-color in the different sections of your page, you can also affect background-color.
Maps

Visit Google Maps and type into the search bar any place in the world where you would like to visit!

Click on the drop down menu.

Select “Share or embed map”.

Click Embed map.

Click "Copy HTML".

Find the < iframe > tag for the Google Map in your HTML code.

Replace all of the iframe code on your website with the new code that you just copied from Google Maps.

Make any other changes you'd like to the site!

Congrats! Project complete!
