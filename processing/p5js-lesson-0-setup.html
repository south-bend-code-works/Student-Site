Setup the p5 environment

  p5.js   LESSON 0
In this lesson you will:
  1. Learn to use the p5.js editor.
  2. Make a p5.js account.
  3. Learn to use the p5.js reference material.

Background Info

Processing — the parent of p5.js — is a coding environment designed for artists and musicians with little to no experience programming. p5.js is the JavaScript version with the added benefit of running natively in the browser. The online editor is easy to use right in the browser with nothing to install. If you are new to writing code, we’ll get you making something awesome by the end of these lessons.

Getting Started

Head over to p5js.org.

For this series of lessons in learning Processing, you'll need an account on p5js.org. Click on Editor, and then create an account.

Navigating the Editor

The main window you see is the editor. This is where we will be writing all our code.
Down below is the console. This is where any errors or messages will show up.
The circular play button will run our code, and the stop button will stop it from running.
The title is auto generated - usually a combination of a couple random words. Right now my title is grave afternoon. You can change this by clicking the pencil icon.
The file menu allows us to create New sketches, Save the current sketch, Open other sketches, and look at Examples.
The edit menu allows us to Tidy up the code, which can also be done with Shift + Tab. We can also use Find, Find Next, and Find Previous to quickly find different pieces of code.
The sketch menu allows us to Add files, Add folders, Run, and Stop the code.
The help menu allows us to access Keyboard Shortcuts, the Reference material, and the About page.
Clicking on the Gear Icon opens the settings menu.
General Settings allows us to change the theme, text size, and turn Autosave and Word Wrap on and off.
The Accessibility tab allows us to turn on and off line numbers, and some accessibility features.

Exploring the Reference Page

The reference page tells us all about every bit of JavaScript code we can use in processing.

Click on background() to see a specific page.
You can press Edit to change the examples to test out how the code works. Press Run to see the results again.
Scroll down to read the Description and learn about how the function works.
Further down is the Syntax, where you can see exactly how to write the code.
After Syntax is Parameters - the values you can pass into the function, and what each one does.
