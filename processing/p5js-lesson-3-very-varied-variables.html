Let There Be Variables

So let’s define them by first declaring they exist.

Javascript is rather permissive when it comes to variable declaration, but for our purposes, we are going to declare them at the very top of our script. Place let x, y, w, h; on the very first line. (Omit one of them for circle(), if that’s the route you took.)
While it’s often a good idea to give variables values right when we declare them, since these are going to be set to environment variables, we have to wait until setup().
There is more than one way to declare a variable in JavaScript. The three ways are with let, const, and var. Each behaves slightly differently. For our purposes, we will just be using let throughout the curriculum. const is used for variables that will be assigned once and not reassigned - a constant variable. let signifies that the variable may be reassigned or changed. var is another catch-all variable definer. There are no restrictions as to what can/can't be done with a variable defined using var, and var should only be used in certain edge cases. For further reading, you can check out this article for a more indepth explanation of let vs const vs var.

At the end of the body of setup(), put x = width/2; on its own line. This is called assignment. Since it is the first assignment, we call it initialization.

We want this circle to be that yellow color from Lesson 2. If you recall, the code appeared like so fill(246, 220, 94);. Well-named variables make programming easier to understand and write. From now on, instead of just putting numbers in for colors, let’s use variables that are meaningful even when a single number will do.

Put let black = 16; below the declaration of yellow.

Now we can set the stroke color to be almost black without having to remember the exact number. We can do the same for the sky blue or whatever color you would like the background to be: let skyblue = [100, 200, 250];

Though black and yellow are both being used as colors, they are both quite different under the covers. Notice that when there are three numbers, we have to place it inside these square brackets [ ], but we didn’t when there was just one. We’ll get into that in a later lesson.
When You’re Smilin’

Because there is a lot of rather specialized code for making a smiley face, rather then walk through it step by step, place this code in the body of draw():
