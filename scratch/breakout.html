Breakout

In this lesson you will:
  1. Build a classic game of breakout.

This is the classic game Breakout, with Watermelons!!
Control the paddle with the right and left arrow keys.

Let's get started!

Open the starter project !

Log into your Scratch account.
  • infoHow to sign in

Title your project.
  • infoHow to add a title
    1. Look at the top of the screen for this text box.



    2. Delete the word "Untitled".

    3. Type in a new name for your project.
Sprite

The cat sprite that you can see is the Scratch mascot. Let’s get rid of it, by right-clicking and then clicking ‘delete’.

Add a paddle.

Add a ball or object that will bounce on the paddle.

Add 1 watermelon.

Add a backdrop.

Paddle Movement

Get the paddle to move left and right using the arrow keys.
  • infoI need a hint
Bouncing Ball

First, get the ball to always move using a forever loop. If the ball is on the edge of the page, get it to bounce back.
  • infoI need a hint

You'll notice that the ball just moves horizontally across the page. To get the ball to move in all directions, add this block under your Event:

Change "90" to "45" degrees.

To get the ball to look like it's bouncing off of the paddle, use a Sensing block.
Your code should read: if the ball touches the paddle, turn 90 degrees.
  • infoI need a hint
Watermelons

When the game starts, when the green flag is clicked, the watermelon should show.

When the ball collides with a watermelon, you want the watermelon to hide.
Score Variable

A variable is a place to store data that can change, like a score.
To create a new variable, click on the ‘Variables’ tab, and then click ‘Make a Variable’.

Type ‘score’ as the name of the variable.
Make sure that it is available for all sprites.
You’ll then see lots of code blocks that can be used with your score variable.

You’ll also see the score in the top-left of the stage.

When a new game is started (by clicking the flag), you should set the player’s score to 0
  • help I need a hint

If the ball collides with the watermelon, change score by 1.

Run the program again. Does the score change?
Duplicate Watermelons

Make the game more interesting by duplicating the watermelon.

To duplicate a sprite and its code, right click on the sprite and select "duplicate".

Position 5-6 watermelons in a line across the top of the screen.
Ball + Paddle Logic

If the ball misses the paddle, the player should lose a life. Let's create another variable called lives.

When the game starts (when the green flag is clicked), set Lives to 3.

If the ball falls below the paddle, change Lives by -1.
  • help I need a hint
Functions

Let's add a function to check if Lives = 0 and end the game.

Go to the My Blocks section and click "Make a Block".

Name the block checkLives.
  • help I need a hint

Now under the define checkLives block, add code so that if Lives = 0 then, stop all.

Add a checkLives block to your ball's script that checks if the ball falls below the paddle.

Save your project

Test your game until you are happy with it.
Challenge

Add a Game Over Screen (when Lives = 0) and You Win Screen (when a player hits all of the watermelons).

Congrats! Project complete!
