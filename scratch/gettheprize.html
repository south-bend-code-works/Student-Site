Coding Movement

We want your character to be controlled with the arrow keys.
Each game starts with an Event.

Events start your code or a set of actions.
Find the Event that you will use to start the whole game.

We want to use loops so that if the left arrow key is pressed, then your character points in the left direction and moves 10 steps.
  • help I need a hint

Once you get your character to move left, we need it to move right, up and down.

Try and write similar code to make your character move up, down, and right with the arrow keys.
Add an enemy

Find the Event that will start the game when the green flag is clicked .

Look in Motion to make your enemy forever point towards your main character and move 10 steps.

To do this, we will use a forever loop. Loops repeat code as many times as we want. A forever loop happens for as long as the game is running.

If the enemy touches the player, the game should end.

See if you can combine an Event, forever loop, if then statement, sensing block, and stop all to make the following sentence true:

When the green flag is clicked, if your enemy is touching your main character then stop all, forever.
Loops

Loops are pieces of code that let us repeat something.

We can repeat some code a specific number of times.
Try and find the repeat 10 block.

We can also repeat code forever by using a forever loop.

The last type of loop is a repeat until loop.
This repeats the code inside it until something becomes false.
