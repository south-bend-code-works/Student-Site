Drawing with Loops

In this lesson you will:
  1. Use loops to create a variety of different shapes and designs.

In this project, you will create program that draws different shapes using loops.

Use the arrow keys to move the sprite between drawings.

Let's get started!

Open the starter project!

Log into your Scratch account.
  • infoHow to sign in

Title your project.
  • infoHow to add a title
    1. Look at the top of the screen for this text box.



    2. Delete the word "Untitled".

    3. Type in a new name for your project.
Sprites

The cat sprite that you can see is the Scratch mascot. Let’s get rid of it, by clicking the blue x in the corner.

Pick a small sprite to be our drawing sprite, or draw a new one.
  • infoHow to add characters

Save your project
  • infoHow to save your project
    Click "File" in the top-left corner



    Click "Save Now"



    Your project is now saved!
Stage Backdrop

Add a backdrop.
  • infoHow to add a backdrop
    At the moment, the stage is white, and looks pretty boring! Let’s add a backdrop to the stage, by clicking the blue icon on the right side of the screen.

    Use the picture below as a reference, if needed.

You can type instructions for the user on the background, or leave it blank.
Sprite Control

To start with, let’s make the sprite move with the arrow keys.

Use a when flag clicked, forever loop, four if statements, the key __ pressed block, change x by and change y by blocks.
  • help I need a hint
Drawing

We need to add the Pen extension. Click on the add extension button in the bottom left corner.

You should see Drawing as an extension. Click on it to add the Pen blocks to your project.

Let's set the pen color and put the pen down when we start.

Use a when flag clicked, and two pen blocks to set the color and put the pen down.
  • infoI need a hint
    You'll need these blocks.
Loops

Now let's add some loops.

We can start with a simple shape like a square.
Get a loop that happens 4 times, and inside, tell the sprite to go forward 50 and then turn 90 degrees.

Let's connect this loop to a keypress. Add the when __ key pressed event and pick what key you'd like to use.

Test it out! The square draws really fast!

We can slow it down by adding a wait block.
More Loops

Duplicate the square code, and see if you can change it to draw a triangle.

How many times should the loop happen? How many degrees does the sprite need to turn?
  • infoI need a hint
    You'll need to loop 3 times, and turn 120 degrees for a triangle.

What about a pentagon? Try making a pentagon loop and connecting it to another keypress.
  • infoI need a hint
    You'll need to loop 5 times and turn 72 degrees.

What other shapes can you make?

How about a star? Try turning 144 degrees 5 times.
Tinker Time!

Can you draw a circle using a loop?

What other patterns can you come up with?
Play with different loop numbers and degrees turned to see what all you can do!

Try using the stamp block to spice things up!
