Lemmings - they follow blindly

In this lesson you will:
  1. Create an avoidance game - keep the mice from falling in the hole!

You will draw paths for the mice to travel to avoid hole and get to the door.

Let's get started!

Drawing

Add the pen extension by clicking on the extension button and selecting pen.

Click on the pen sprite and add code to change the pen color when the flag is clicked. Make the color the same blue as the ground.
  • infoNeed a hint?

Add code to make the pen follow the mouse pointer.

You'll need a forever loop and a go to mouse pointer block.

Add more code to put the pen down if the mouse is down and else pick the pen up

Test your code!

Save your project
  • infoHow to save your project
    Click "File" in the top-left corner



    Click "Save Now"



    Your project is now saved!
Clones

Let's add a series of mice that fall and follow each other like lemmings.

Click on the mouse sprite, and add code to tell the mouse to hide and then forever create a clone of myself and wait 3 seconds.

Add code to tell the mouse when I start as a clone to show and fall until touching color blue

At this point, the mice should be falling and piling up on the screen.
Mouse Movement

Once the mice hit the blue ground, they should move forward until they hit the door or fall into a hole.

Add code to the when I start as a clone script to move forward and change costumes every 0.1 seconds so that it looks like the mouse is walking.

Repeat all the above code until the sprite touches an edge
  • infoHint
Refining Movement.

Add another loop into the mouse clone script to tell the mouse to tell the mouse to move up 2 until not touching blue.
  • infoHint

Test out your code!
Getting to safety.

Add a variable called score.

Add code to the mouse script that will increase the score by 1 everytime a clone touches the door.

Add code to the mouse script that will set score to 0 when the green flag is clicked.
  • infoHint

Add code to tell the clone to play a sound and delete this clone when it touches the door.

Test your project, does it work?

Save your project
