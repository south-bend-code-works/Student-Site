About Me

In this lesson you will:
  1. Create different sprites to learn all about me!

In this project, you will create a Scratch project that describes you!

Let's get started!

Open the starter project !

Log into your Scratch account.
  • infoHow to sign in

Title your project with your name.
  • infoHow to add a title
    1. Look at the top of the screen for this text box.



    2. Delete the word "Untitled".

    3. Type in a new name for your project.
Characters

The cat sprite that you can see is the Scratch mascot. Let's get rid of it, by clicking the x in the corner of the cat icon.

Add characters that represent you to your project.
For example, you could add sprites of your favorite sports, food, games, etc.
  • infoHow to add characters

Add at least 3 sprites that represent you + your favorite things.

Save your project
  • infoHow to save your project
    Click "File" in the top-left corner



    Click "Save Now"



    Your project is now saved!
Stage Backdrop

Add a backdrop.
  • infoHow to add a backdrop
    At the moment, the stage is white, and looks pretty boring! Let's add a backdrop to the stage, by clicking the blue icon on the right side of the screen.

    Use the picture below as a reference, if needed.

Animations

Brainstorm which animations you want for each sprite
For example, you can program your sprite to spin, talk, change different colors, etc.
Sample Animations:

Say Something

Animate Talking

Add Sound

Move Using Arrow Keys

Jump

Change Color

Each animation starts with an Event.
An Event starts your code or a set of actions.

Use at least 3 different types of Events in your project.

Save your project

Congrats! Project complete!
