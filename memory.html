Memory Game

In this lesson you will:
  1. Create a memory game where you have to memorize and repeat a sequence of random colors!
Let's get started!

Open the starter project!

Delete the cat sprite so that your project is empty.

Choose a character. Your character doesn't have to be a person, but it needs to be able to show different colors.

In your game, you'll use a different number to represent each color. Example:
  • 1=red;
  • 2=blue;
  • 3=green;
  • 4=yellow;
Give your character four differently colored costumes, one for each of the four colors above. Make sure that your colored costumes are in the right order

To create a random sequence, you need to create a list. A list is just a variable that stores lots of data in order.

Create a new list called 'sequence'.
