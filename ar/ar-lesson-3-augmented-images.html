Augmented Images

 AR  Lesson 3
In this lesson you will:
  1. Detect images in Augmented Reality space.
  2. Place objects on images in Augmented Reality space.
  3. Use ARCore, the SDK for Android AR Development.

Background Info

Augmented Reality uses image detection and recognition to allow the user to create a virtual object on top of only specific objects in the real world.

Helpful Links

Project Setup

Create a new app by following the instructions in Lesson 0 - Setup.

Import the image(s) you want your app to recognize into the project by dragging them into the Assets folder.

Select the image(s), then right click and select Create -> Google ARCore -> Augmented Image Database.

Rename the database something like Augmented Images DB.

Open up the DefaultSessionConfig by going to the ARCore Device and double clicking DefaultSessionConfig.

Under Augmented Images, add the AugmentedImagesDB you just created in the Augmented Image Database slot.

Add an Augmented Image Example Controller script to the ARCore Device.

In the Augmented Image Visualizer Prefab box add the prefab found in Assets -> Google ARCore -> Examples -> AugmentedImages -> Prefabs.

Add a new empty game object named EventSystem.

Add an Event System Script to EventSystem.

Set the drag threshold to 5.

Add a Standalone Input Module to EventSystem.

Build and Test

Build and Run your app.

Test the app using the photo(s) you added to the Image Database.

You may need to move the camera close to the image, and then further away to see the prefab show up.

Adding a Different Prefab

Create an Empty Game Object called Pawn.

Add a new C# script to this called AugmentedImagePawn.cs.

In this script add the following code.
using System;
using System.Runtime.InteropServices;
using GoogleARCore;
using GoogleARCoreInternal;

Add the following two variables.
public AugmentedImage Image;
public GameObject Object;

Remove the start function.

In the Update function add the following code.
if (Image == null || Image.TrackingState != TrackingState.Tracking)
{
Object.SetActive(false);
return;
}
Object.transform.localPosition = new Vector3();
Object.SetActive(true);

Add a Pawn from Assets -> GoogleARCore -> Examples -> Common -> Materials as a child of the Pawn GameObject.

Drag that child into the Object box on the AugmentedImagePawn script on Pawn.

Drag the Pawn object into the Assets folder to make a prefab, and then delete the object still in the Hierarchy.

Create a new script called Augmented Image Controller. Right click in the Assets folder and select Create -> C# Script.

Add the following code to the new script.
using System.Runtime.InteropServices;
using GoogleARCore;
using UnityEngine.UI;

Add a public variable as shown below.
public AugmentedImagePawn AugmentedImagePrefab;

Add two private variables as shown below.
private Dictionary<int, AugmentedImagePawn> m_Visualizers = new Dictionary<int, AugmentedImagePawn>();
private List<AugmentedImage> m_TempAugmentedImages = new List<AugmentedImage>();

Delete the start method.

Add a public void Awake method.
public void Awake()
{
Application.targetFrameRate = 60;
}

In the Update method, add the following code.
if (Input.GetKey(KeyCode.Escape))
{
Application.Quit();
}
if (Session.Status != SessionStatus.Tracking)
{
Screen.sleepTimeout = SleepTimeout.SystemSetting;
}
else
{
Screen.sleepTimeout = SleepTimeout.NeverSleep;
}

Session.GetTrackables<AugmentedImage>(m_TempAugmentedImages, TrackableQueryFilter.Updated);
foreach (var image in m_TempAugmentedImages)
{
AugmentedImagePawn visualizer = null;
m_Visualizers.TryGetValue(image.DatabaseIndex, out visualizer);
if (image.TrackingState == TrackingState.Tracking && visualizer == null)
{
Anchor anchor = image.CreateAnchor(image.CenterPose);
Visualizer = (AugmentedImagePawn)Instantiate(AugmentedImagePrefab, anchor.transform);
visualizer.Image = image;
m_Visualizers.Add(image.DatabaseIndex, visualizer);
}
else if (image.TrackingState == TrackingState.Stopped && visualizer != null)
{
m_Visualizers.Remove(image.DatabaseIndex);
GameObject.Destroy(visualizer.gameObject);
}
}

Create a new empty game object called GameController.

Add the Augmented Image Controller script to the GameController.

Drag the Pawn prefab into the Augmented Image Prefab box on the Augmented Image Controller Script.

Build and Run

Run your app and test to make sure that it recognizes your image and generates the prefab.

Turn off the original prefab by deactivating the Augmented Image Example Controller on the ARCore Device.

You may also want to resize the pawn.

Double click the pawn prefab to open it, then adjust the scale of the pawn model.

Run your app and test until the size is what you want it to be.
Lesson complete! Good job!
