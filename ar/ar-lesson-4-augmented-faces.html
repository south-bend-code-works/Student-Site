Augmented Faces

 AR  Lesson 4
In this lesson you will:
  1. Detect faces in Augmented Reality space.
  2. Place masks on faces in Augmented Reality space.
  3. Use AR Foundation.

Background Info

Augmented Reality uses face detection to allow users to add textures and materials to faces to create filters and masks.

Helpful Links

Docs: AR Core documentation.
AR Foundation: AR Foundation Unity documentation.
Sample Images: Feel free to use these images for your filters.

Project Setup

Create a new 3D Project in Unity.

Open Build Settings, select Android, and click Switch Platform.

Open the Package Manager. Window -> Package Manager.

Install the latest version of AR Foundation Package. (may be newer than pictured)

Install the latest version of AR Core XR Package. (may be newer than pictured)

Open Player Settings from Build Settings.

Adjust the Company and Product name to what you want.

Under Other Settings, enable Auto Graphics API.

Disable Multithreaded Rendering.

Set the Minimum API level to API 24.

Select AR Core Build Settings under the XR Menu.

Click Create, and store the build settings in the Assets folder.

Select XR Plugin Management.

Click the + under the Plugin Providers List.

Select AR Core Loader.

Making Face Filters

Right click in the Hierarchy, and select XR -> AR Session.

Right click in the Hierarchy, and select XR -> AR Session Origin.

Add AR Face Manager to Session Origin.

Right click in the Hierarchy and select XR -> AR Default Face.

Create folders inside Assets named Prefabs, Materials, Textures.

Drag the images you want to use for masks into the Textures folder.

Inside the Materials folder, create a new material with desired texture.

Click the circle next to albedo and select the image you want to use as the texture.

Adjust metallic and smoothness level to get the desired look.

Select the AR Default face in the hierarchy.

Drag and drop the facematerial to Element 0 in the Mesh Renderer component of AR Default face.

Make prefab out of the AR default face.

Drag and drop AR Default Face into the Prefabs folder.

Rename the prefab to something useful.

Delete the AR Default face from the hierarchy.

Select the AR Session Origin in the hierarchy.

Drag the face prefab to the AR Face Manager component’s Face Prefab box.

Build and Run

In the file menu, select Build Settings. Add the open scene to the build, then Build and Run.

Add Multiple Filters

Add a Button to the screen. Right click in the hierarchy and select UI -> Button.

Anchor it to a corner by selecting the anchor button, then pressing shift + alt while selecting a corner.

Make a few more materials using other images (follow the steps from above).

Store all materials in the Materials folder.

On the AR Session Origin, create a new script called Face Controller.

Add the following code under the using UnityEngine;
using UnityEngine.UI;
using UnityEngine.XR.ARFoundation;

Add a require component for ARFaceManager just before the class definition.
[RequireComponent(typeof(ARFaceManager))]

Add a [SerializeField] and then create a new private button.
[SerializeField]
private Button swapFacesToggle;

Add a private ARFaceManager called arFaceManager.
private ARFaceManager arFaceManager;

Add a private integer swapCount, set it to 0.
private int swapCount = 0;

Add a [SerializeField] and then create a new array for FaceMaterials
[SerializeField] public FaceMaterial[] materials;

Create a void Awake method.
void Awake()
{
arFaceManager = GetComponent<ARFaceManager>();
swapFacesToggle.onClick.AddListener(SwapFaces);

arFaceManager.facePrefab.GetComponent<MeshRenderer>().material = materials[0].Material;
}

Create a void SwapFaces() method to switch between faces.
void SwapFaces()
{
if (swapCount == materials.Length - 1)
{
swapCount = 0;
}
else
{
swapCount++;
}

foreach (ARFace face in arFaceManager.trackables)
{
face.GetComponent<MeshRenderer>().material = materials[swapCount].Material;
}

swapFacesToggle.GetComponentInChildren<Text>().text = $"Face Material ({materials[swapCount].Name})";
}

After the closing } for the class, create a new class.
[System.Serializable]
public class FaceMaterial
{
public Material Material;
public string Name;
}

Save the script.

Back in Unity, open the AR Session Origin.

Drag your button into the button box on the FaceController script.

Type in the number of materials you have in the Materials Size box.

Drag each material to a box and give each material a name.

Build and Test!
Lesson complete! Good job!
