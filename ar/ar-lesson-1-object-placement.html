Placing Objects in AR

 AR  Lesson 1
In this lesson you will:
  1. Detect planes in Augmented Reality space.
  2. Place objects in Augmented Reality space.
  3. Use ARCore, the SDK for Android AR Development.

Background Info

Augmented Reality is a technology that superimposes a computer-generated image on a user's view of the real world, providing a composite view of the real world, and the computer generated world.

Helpful Links

Docs: AR Core documentation.

Project Setup

Start with the app you created in the AR Core Setup Lesson. If you haven't completed that lesson, go back and to Lesson 0 - Setup.

Create a new empty game object called Plane Generator.

Add a Detected Plane Generator Script.

In the Detected Plane Prefab box drag the DetectedPlaneVisualizer from GoogleARCore -> Examples -> Common -> Prefabs.

Create a new empty game object called AR Controller.

Add the Hello AR Controller Script.

Drag MainCamera into First Person Camera.

Create a new empty game object called ObjectToSpawn.

Inside this object, create a cube (or sphere).

Add a material to color the item.

Remove the collider from the object.

Resize the object to 0.1/0.1/0.1.

Set the y position to 0.05.

Drag the ObjectTo Spawn to the Assets folder to make a prefab.

Delete the ObjectToSpawn from the Hierarchy.

Drag the ObjectToSpawn prefab into the three Prefab boxes on the AR Controller script.

Add an empty game object named EventSystem.

Reset the transform for EventSystem.

Add an Event System Script to EventSystem.

Set the drag threshold to 5.

Add a Standalone Input Module to EventSystem.

Add an empty Game Object named PointCloud.

Add a mesh filter to PointCloud.

Click the circle to the right of None (mesh) to select the mesh of a cube.

Add a mesh renderer to PointCloud.

Select PointCloud for Element 0 in the Materials list (click the circle and search for point cloud).

Add a PointCloud Visualizer Script to PointCloud.

Building to Android Device

Computer Steps

Plug in your Android device via USB.

Click Build and Run in the File menu.

Choose where to save the .apk and name it appropriately.
The location is not very important. You can leave it in your Unity project folder.

You may need to allow data to be transferred from the Android to the computer when the dialog box appears.

The app will take a few minutes to build and run.

Android Device Steps

Test your app.

Click the dialog box to allow the app to record pictures and videos.

Move the camera around to detect planes.

Try placing objects by tapping on the planes identified by the app

Further Tinkering

Try swapping out the ObjectToSpawn prefab with a different object. Just repeat the steps, but use a different 3D object inside the prefab.
Lesson complete! Good job!
