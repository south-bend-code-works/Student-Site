Falling Fruit

Create a game all about catching fruit and avoiding basketballs!

What you will make

Avoid the basketballs and catch the apples.

Open the starter project !

Coding Movement

The bowl will move left and right to catch the fruit. We don't want it to turn upside down though!

Find the set rotation style Left-Right block.
  • help I need a hint

Add code so that if the left-arrow key is pressed, the bowl points in direction left and moves 10 steps, forever.

Add similar code for the right arrow key and right movement.
Falling Fruit

Go to your fruit sprite.

We want it to start at the top of the screen, with a random x-coordinate.

Use a go-to block and a pick random block to make the fruit start at a random x value between -230 and 230, and a y value of 180.

Then we want the fruit to fall continually, so we need to change y by a negative number.

Finally, when the fruit either touches the bowl, or reaches the bottom of the screen, we want it to go back to the top.

Create two if-then blocks.

One should check if fruit touches the bowl then go back to the top.

The second one should say if the y-position of the apple is less than (<) 180, to go back to the top.
  • info Hint

Don't forget that all the code for the fruit should happen forever.
Score

Add a variable called score.

Set score to zero when the game starts.

Increase score by one whenever the fruit touches the bowl.
Non-Fruit Object

Now we will code the other object to fall continuously, decrease score by 1 if touching the bowl, and go back to the top if touching the bowl or the bottom of the screen.
