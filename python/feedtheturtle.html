Feed the Turtle

In this lesson you will:
  1. Create an interactive game about feeding a turtle.
  2. Incorporate a score feature.
  3. Add an enemy turtle!
Background Info

We can use multiple turtles in a single project, each one doing a different job. In this project, we will use four different turtles for different things!

Adding an Enemy

Begin by adding one more turtle, this one named enemy.
Change enemy's settings like we did for food and player. Set the shape, color, set the speed to 0, and pick the pen up.
We also want the enemy to start near the top of the screen. Tell the enemy to goto a place near the top of the window.

Let's add a function to check and see if the enemy turtle is touching the player turtle. Call this function checkEnemy().

This function should call global score, and check if the distance from the player to the enemy is less than 20. Then, Set the score to 0, and tell the scoreWriter to write the new score.

Now we just need the enemy to move on its own. We will define a function for this called enemyMove().

The enemyMove function should have an infinite loop while True: that moves the enemy forward, sets the enemy heading to enemy.toward(player) , and calls checkEnemy.

Now we just need to tie this function to a keypress. Bind it to the space key.

Now play the game. Press space to activate the enemy!
