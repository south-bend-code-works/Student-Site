Designing: A flowchart of TicTacToe

Game AI

Part of this program is working together with the AI to build the game board, so that it can look at the board and decided where and what kind of spaces to move to.

Think of it like this

The AI algorithm will go through these steps:

1. Try to find a move that the computer can make to win. Then take that move. If not, go to step 2.

2. See if there's a move that the player can make that will cause the AI to lose. If so, block that move. If not, go to step 3.

3. Check if corner spots are open, then move there. If not, move on to step 4.

4. Check if the center is open. If not, move on to step 5.

5. Move on to any of the side spaces. There are no more steps because after steps 1-4, the side spaces are all that is left.
