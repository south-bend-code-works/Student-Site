Writing with Turtles

In this lesson you will:
  1. Create writing with Python turtles!
Background Info

We can have our turtles do more than just draw shapes. Turtles can write text on our screen as well.

Helpful Links

repl.it - an online interpreter that we will use for writing our Python code.

Let's get started!

First steps

Open repl.it

Name it TurtleWriting

Make sure you create this new Repl using Python (with Turtle)

Make a new turtle named t using turtle.Turtle()

Letâ€™s have our turtle draw some text.

By default the font is the Arial typeface, size 8, and normal.

We can change how the text is aligned, using the align keyword.

The options for align are right, center, and left.

We can also change the font the turtle uses, using the font keyword.

Font needs a triple, as shown below. You need to tell it the font name, font size, and font type.

Some font options are Arial, Times, Courier, and Calibri. (there are lots of others also)

The fonttype options are bold, italic and bold italic.
Bonus: Hiding and Showing

You can make the turtle hide and show easily!

Use t.hideturtle() or t.ht() to hide the turtle.

Use t.showturtle() or t.st() to show the turtle again.
