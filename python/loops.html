While Loops

So for loops are very useful if we know how many times we want a bit of code to happen. But what if we're not sure how many times something needs to happen? We can use a while loop for this.

While loops are just like for loops, they repeat code, the difference is that while loops use a boolean expression to know when to stop.

A Boolean expression is a logical statement that is either TRUE or FALSE.

See the example below:

Boolean Expressions

The boolean expression above, x < 5, tells the while loop when to continue and when to stop.

Each time the loop starts over again it checks to see if x is still less than 5. As long as x is less than 5, the loop runs again. As soon as x is not less than 5, the loop stops.

Endless Loop

We can easily run into some trouble here though.

For example:

Do you see the problem? We've just created an endless loop! (you can use the stop button at the top of your screen to stop this.)

Because x never changes value, it's always 0, and 0 is always less than 5, so this loop keeps going forever! (or until we stop the terminal.)

So it is very important when using while loops, to be sure that the boolean expression will become false at some point.
