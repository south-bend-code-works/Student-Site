Background Info

Print is a built-in python function, which means it's already defined for us. To use the print function, we just need to put whatever we want to print inside the parenthesis. If we are printing strings, remember to put quotes around them.

Variables are like empty boxes to store information in that we want to use later. We can name them almost anything, but it is helpful to give variables logical names that describe what they are holding. It's like labeling boxes!

A function is a group of code. Using functions is a quick and easy way to group a bunch of code together to be able to use whenever we want.

Naming Conventions

When we name variables, we will follow a naming convention called camelCase. This just means that the first letter of a variable name will be lowercase, and the first letter of every other word in the name will be capitalized. For example:

firstName

lastName

streetAddress

thisAwesomeName

Variable names must start with a letter or underscore, and cannot contain anything other than letters, numbers, and underscores.
