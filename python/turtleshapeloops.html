Starry Night

Let's draw a starry night using for loops.

We can start by changing the background color to navy.

Now we can change the turtle color to yellow or silver, and draw a small star.
t.color('silver')

Now we just need to move the turtle to a new spot and draw another star.

t.goto(random.randint(-300,300), random.randint(0,300))
We also need to import random up with our turtle import.

But the turtle is still drawing, let's pick up the pen before moving, and put it down after moving.

Now we can take all of that code (the star and the movement), and put it inside a for loop to happen many times!
