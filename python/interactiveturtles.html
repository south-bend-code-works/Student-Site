Interactive Turtles

In this lesson you will:
  1. Make controllable turtles!
  2. Make an etch-a-sketch using turtles.
Background Info

We can interact with our turtles while our programs are running.


Helpful Links

repl.it - an online interpreter that we will use for writing our Python code.

Let's get started!

To make turtles interactive, we will need to use the screen object, and define some functions to respond to keypresses.

Start a new repl for Python with Turtles and import turtle.

We can create a screen object with the command turtle.Screen

We need to assign it to a variable so we can call on it later.

We can change the background color using this variable now.
Interacting with your turtles

Now let's make a turtle!

Feel free to change the color and shape of your turtle.

Go ahead and turn the speed up

We'll need a couple of functions for the turtle movement. Let's make a function for each direction we want the turtle to move in.

This will point the turtle "t" up and move it forward 10 pixels.

Go ahead and make three more functions called "down", "move_right", and "move_left".

Helpful Hint: 0 is facing right, 90 is up, 180 is left, and 270 is down.

Now we will connect our functions to keypresses.

The Screen object has a method called onkey(function, key) that binds a function to a keypress.

This binds the up() function to the Up arrow key being pressed.

Add bindings for the other three functions. Use "Down", "Right", and "Left" for the other keypresses.

Now all we need to do is tell the screen to listen! Add this method at the end.

What if we want to pick up the pen and put it down with another keypress? Let's add that feature!

Make another function that will pick up the pen if it's down, and put it down if it's up.

Hint: You'll need to use t.isdown() to see if the pen is down.

Bind this function to the spacebar!

You can also make other functions, like changing the pen size or color, and bind them to other keypresses.
