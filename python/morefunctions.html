Syntax

We've already written quite a few functions, but let's review the syntax one more time.

All functions start with def, followed by the function name. After the function name comes a set of parentheses, surrounding the parameters, which are separated by commas. The second parenthesis is followed by a colon.

Anything inside the function must be indented underneath the function definition.

Parameters and Arguments

Parameters are the variables that let you pass information into the function.

When you call a function and pass values into it, those values are called arguments.

For example. Above, the parameters are called parameter1 and parameter2.

If I called the function functionName(4, 6), then 4 and 6 would be arguments, the values that I'm putting into the function.

Output and Return

There are a few different ways to give information back to the user. So far we have used print() a lot to print information out onto the screen. If we want to give the user back a value that they can store in a variable, or use immediately for something else, we need to use return.

Look at the examples below.

If I called add(4, 5), I could take the value 9 that is returned and use it again.

However, if we used print() instead of return, we couldn't use the value.

**Note the difference in color also. Printed values are white in the terminal, while returned values are => green.**
