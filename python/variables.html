Background Info

There are quite a few different data types. For this lesson, we will be looking at four of them. We will study more of them later on.

Python is a loosely typed language, which means that when you define a variable, you don't need to tell the interpreter what kind of data is being stored in the variable. The Python interpreter will just figure out what kind of variable each one is when it runs.

Strings

Now we get to the fun stuff!

Now we get to the fun stuff. Strings are a variable type you've already seen a bit and worked with. Let's go deeper.

For information to be considered a string, it needs to be surrounded by quotation marks. You can use either double-quotes “ or single-quotes ‘, but the quotes on each end have to match!


You try it.

Define a couple of strings

Notice that anything you put inside quotes becomes a string, even if it otherwise would have been something else.

But what if you want to put a quote inside a string?

If we put a quote inside a string, the python interpreter thinks that the quote is ending the string, and we get an error!

Escape Characters

To solve this problem, we can use something called an escape character.

Simply put, we can use a backslash \ in front of any special characters in a string to tell the Python interpreter to ignore the character.

There are a few other escape characters we use.

String Indices

Another cool feature of strings is that they are iterable, which means that you can count how many items are inside a string and you can call a for loop (we will learn about these later) on a string.

More simply, every character inside a string has a number telling you where in the sting the character is. That number is called an index.


For example:

The string My name is Kristen has 18 characters. These each have a number starting at 0.

**In computer science we almost always start counting at 0 instead of 1!**

So if we ask the Python terminal to give us the character at index 6 of the string, we will get an e.

Type Conversions

So we've looked at 4 different types of data.

Now what if we want to turn one kind of data into another?

This process is called type conversion. Sometimes Python does this automatically for us.

Here, the Python interpreter converted the integer 8 into a string so that it could be printed.

But if we wanted to concatenate a string and an integer, we would need to convert the integer into a string first.

The str() function is a built-in function that lets us convert other data types into strings

We can also convert strings into integers if the strings contain only numbers.

If the string contains anything other than numbers, we will get an error.

We can also turn strings or integers into floats.
