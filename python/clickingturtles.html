Clicking On Turtles

In the last bit, we added a generic command to make the turtle move to our mouse pointer when we clicked the screen. But what if we want to do something only when the turtle is clicked?

For that, we will use the onclick method of the turtle itself, instead of the onclick method for the screen.

Let’s make the turtle go somewhere random when we click on it.

Import the random module at the start of your file.

Now define a function called randPos(x,y).

Inside that function, tell the turtle t to go to a random x and y, between -200 and 200.

Now we will bind that function to t being clicked.
