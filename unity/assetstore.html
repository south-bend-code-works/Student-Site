Unity Cheat Sheet - Asset Store

Forgot stuff? It happens!

The Asset Store is an extremely useful tool because that's where we find all of our models, animations, sounds, and more!

While the Asset Store is useful, you can always create your own assets. Models can be shaped with Blender, animations can be crafted in the game, and so much more. But, while there are benefits to creating your own assets (hello copyright free!), they are extremely time consuming.

Asset Store

To get to the Asset Store, look for its tab above the Scene view.

Go to the search bar and type in what you're looking for, "characters" or "animations".

For this class, we want to make sure you are using free assets. There are many high quality assets out there for free, so there is no reason for you to pay for them.

Click on the drop down, and select Price (low to high)

Once you find an asset you are looking for, click on it.

On the right-hand side you will find the red download button, click on that.

It will then begin to download. Once it is done downloading, Unity will ask you if you want to import it. You'll probably want all of the assets it suggests, so just hit Import all.


If you go to the Project View after this is done, you'll see a new folder with the name of the folder you just imported.
