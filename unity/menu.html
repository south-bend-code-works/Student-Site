New Scene

Great work with your game so far!

Now it's time to add a little bit of polish, a main menu. Make sure to save your current scene before you do this.

Then go to File → Save and call it MainMenu.

This scene will be the UI for our main menu!

To start, make the scene 2D by pressing the 2D button on the Scene view.

Try to think of this menu as place cards in front of a camera. The 3D space is still there, it's just being covered, or intercepted by the menu.
Button

In the Hierarchy, go to Create → UI → Button.

Buttons have things like an OnClick function by defult, so you won't have to manually code this.

The button has various features you can edit:
  1. Text on the button(the text is a child of the button)
  2. Color
  3. Hovering color
  4. Clicked color

If you scroll down on the Button's components in the Inspector, you'll see a box that says On Click ().

This box allows the button to call a function when it is clicked. In this case, we want it to call a function that loads the first level. To do this, we need to make that function on a script. Let's make a new empty GameObject called MenuManager that will hold the script.

In the Hierarchy, press Create → Empty.

Now go onto the MenuManager empty GameObject and click Add Component → New Script.

Name it MenuManager. Open it up in the code editor.

Remember that we are using this script to make a function that loads a level. In order to load a level, we need to add the SceneManagement namespace.

Now let's get into the script proper. Erase the Start and Update functions and make a new public function called .LoadLevel.

The function needs something to load. Let's give it an argument.

A “string” is another variable type! It can contain numbers and text (like a level name!)

Great! That's really the whole script. Now we just have to assign it. Go to the StartButton GameObject in the Inspector and click the plus button in the OnClick () box. Drag the MenuManager GameObject into where it says None.

Click the drop-down → MenuManager → LoadLevel(string).

The box that appears is where we put the string of the level. Put the name of your first level there.

Save the scene you're on and then play the game and try the button! It should load your first level. Make sure the scene you're trying to load is in the Build Settings!
Text

Text boxes are good for titles and instructions. Text is even easier to make than buttons!

Hierarchy → Create → UI → Text

The box can be moved around on the canvas with the normal movement tools. If you want to change the size, use the RectTransform tool (the furthest right on the movement tool). The text can be entered into the Text component in the Inspector.

This is when you can use your highly refined skills of worldbuilding, and make a great title for your game.
Menu Background: Solid Color

You can make the menu have a solid background by editing the Main Camera in the Inspector.

Set Clear Flags to Solid Color

Menu Background: Background Image

Create an image (Piskel or the GIMP software installed on your computer are great tools to create your own background image)

Drag that image into the Project view to import it

Click on the image in the Project view to open its import settings in the Inspector.

Change the texture type to Sprite (2D and UI).

We're going to make that the texture of a UI image component. Create one:

Create → UI → Image

In the Inspector for the Image, drag the sprite you imported to the Sprite variable.

Click preserve aspect to make sure that the image isn't stretched strangely. You can then click native size to make the image object and the sprite the same size. Use the rectangle transform tool to resize it! Finally, make sure that the image is the first child of the canvas. This will make it render first (the text and button will render in front of it so the player can see them!
