Making the Terrain
Let's jump into Unity

Create a new game with instructions from Setup

Click Create in the Hierarchy

on the left-hand side, press 3D Object → Terrain

Now you have your first world. Let's jazz it up!

Adding Textures

Click the Terrain in the Hierarchy (right) to open it in the Inspector (left).

Click the dropdown menu that says Create Neighbor Terrains, and select Paint Textures

Find the texture you want to be the default for the terrain in the project window: Scroll through the options, or search for "grass”, “rock”, or “mud”. Click the one you want, then press Enter.

For each Terrain layer you create, use one texture. This isn't a limit on your worldbuilding. To add more textures, hit the Edit Terrain Layers → Create Layer button again and add another in the same way.

Painting the Terrain

You can paint the terrain with the mouse in the Scene view.

Brushes: the shape of the brush you're painting with.

Terrain Layers: click the one you want to paint with.

Brush size: the size of the brush.

Opacity: the transparency of the texture. You can layer textures with different opacities.

Not finding the terrain layers you want? Look for more terrains in the Asset Store

Building moutains out of molehills

Changing the Height of the Terrain

Click the dropdown menu that says Paint Texture, and select Raise or Lower Terrain.

The Raise/Lower terrain tool allows you to change the shape of the terrain. You can make mountains, valleys, hills, etc. Paint with it just like the texture painting. You can adjust the brush, brush size and opacity just like the other tool.

Click: raise the terrain.

Shift + Click: lower the terrain.

Opacity: how much you change the size. Keep this low for small adjustments or high for big changes in the landscape.

You can also use the Set Height option in the drop down to set the height of your terrain to a specific value.


The Smooth Height option can be used to smooth out jagged edges and points.

Now let's add those trees in

The Tree tool is second on the Terrain tool bar.

Add a tree model:

Edit Trees → Add Tree

This will open a pop-up window asking for a tree prefab.

In this pre-fab, there are three options: broadleaf, conifer, and palm

Go to Assets → Standard Assets → Environment → Speedtree → Your trees

Find the tree prefab, which is the one with the icon featuring a 3D tree and a grey background.

(Or drag in your own custom tree if you aren't using the Standard Assets)

Drag that prefab into the box asking for the tree prefab in the Add a Tree menu.


Trees are painted just like the terrain. There are various new options:

Tree Density - how many trees you're placing in the painted area.

Tree Height - how tall the tree is. This can be random and the range can be dragged.

Random Tree Rotation - having random tree rotation is good for variety.


Recommendation: Trees are pretty resource intensive. Having too many can really slow your game down or even lead to a crash. Be careful.

Adding Water

Water is an amazing way to make your terrain look more realistic.

Go back into your terrain, adjust your brush.

Create an indent in the terrain where you want your water to be.

Does it make sense in your story to have just one pond? To have several? A river?

Get the water from: Standard Assets → Environment → Water → Water → Prefabs

From there, drag it onto your scene and adjust its position and size with the menu

Can't remember how to move it? check out the Movement Tools Cheat Sheet!

Lighting

To edit lighting, select the Directional Light that came with your scene in the Hierarchy.

Rotation - Can edit this in the Inspector's transform component. This is the angle for the light. Directional light's positioning and scale do not affect their lighting.

Intensity - How bright the light is. This is on the Light component in the Inspector.

Color - The color of the light. Can be adjusted on the Light component in the Inspector.
