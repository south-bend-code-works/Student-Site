JQuery Getting Started

jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML. It is free, open-source software using the permissive MIT License.

Let's get started!

Get JQuery CDN

Google jquery cdn

Click on the first link that says jQuery CDN

In the first section, JQuery 3.x, Click on the minified version.

There will be a pop-up window. Copy the indicated code.

Go back to Atom and paste the code in the < head > right above your personal < script > tag.

Good Job! The JQuery CDN is in.

JQuery is a very useful tool when coding in JavaScript. In future projects use this JQuery Cheat Sheet to assist you with finding the right event to use.
