Lists, tuples and dictionaries are three more data/variable types in Python.

All three are iterables, that means that they hold multiple pieces of information and can be looped over and over.

Lists

Syntax

Square brackets [ ] denote a list.

Lists are mutable, you can define them, and then change the length of the list, add items, or replace items inside the list.

You can also add an item to a list by using the keyword append.

To access a particular item in the list, use the index and square brackets to access a particular element.

Lists generally contain the same type of data (or homogenous data).

Tuples

Syntax

Parentheses ( ) denote a tuple.

Tuples are immutable, meaning once you set their elements, you cannot change them.

Tuples often contain different types of data (heterogenous data).

Dictionaries

Syntax

Curly braces { } denote dictionaries.

Dictionaries are sets of key : value pairs. Think of them like a phone book. The key is a name, and the value would be a phone number.

Each part of the data pair is separated by a colon, and each data pair is separated by a comma.

Each pair is accessed by the key, rather than by a numerical index.

You can delete items from a dictionary just like you can from a list.
