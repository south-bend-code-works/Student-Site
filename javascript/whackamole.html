Adding Images

Each box has 4 different images, appearing at different times depending on how the user interacts with your game.

Look up 1 image for each of the following categories AND save the images to your folder:

Image #1- Default Image (ex. pile of snow)

This is the image that appears when the page first loads. The user has not clicked or done anything on the page.

Image #2- Hover Image (ex. question mark)

This is the image that appears when the user hovers over each box (aka the user places their cursor on each box but does not click).

Image #3- Click - Good Image (ex. penguin)

This is the image that appears when the user clicks on a box but has not lost the game. Instead, a good image pops up when the user clicks.

Image #4- Click - Bad Image (ex. yeti)

This is the image that appears when the user clicks on the wrong box and will now lose the game.
