Adding Functions

We will need three onclick functions to determine what happens when the user clicks the start, stop, and reset buttons.

We will start with the stop button, as it has the simplest function.
After your variables, add this code:
stopbutton.onclick = function(){
clearInterval(Interval);
}

Next let's do the reset button.
When we reset, we need the timer to stop, all the time variables to go to 0, and all the displays to update to 00.

Now for the start button.
Add code similar to the stop button.
startButton.onclick = function(){
clearInterval(Interval);
Interval = setInterval(startTimer, 10);
}

This time we added the Interval = setInterval(startTimer, 10) line of code.
This line tells Javascript to run the startTimer function every 10 milliseconds.
For more info about the setInterval function, you can read up here.
