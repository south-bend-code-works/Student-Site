Unity Cheat Sheet - Error Messages

So you've got an error. Unity won't let you run your game because of it. Not to worry! Unity will tell you what's wrong. Here's how to read the messages:

Go to the console (it's a tab above the project window)

In the top-right of the window, you'll see three toggles. These toggle debug messages (the speech bubble), warnings (the triangle), and errors (the stop sign). The errors are what we're worried about.

If you look in the window itself, you'll see any errors you have. They all have the same structure:

In this error, we see we have an error in the CollectableScript. It is on line 25. The error code is CS1002 (can be useful for Googling!). The error message itself is “; expected”. If I go to line 25 of the CollectableScript…

I'll see that I forgot a semicolon. Visual Studio also highlights this with a red line.

Try your best to solve errors yourself.

You'll learn more this way (development involves a lot of Googling). Googling your error codes, if you're not sure what they mean, can help.

If you're truly stuck, ask a nearby friend or an instructor. Good luck!
