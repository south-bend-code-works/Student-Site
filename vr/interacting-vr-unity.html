Interacting With Objects

In this lesson you will:
  1. Add objects to your scene
  2. Use scripts to make objects interactable

What You Will Make

There are a couple simple scripts you can add to objects to make the player able to interact with them. In this lesson, we will add some built in prefabs, and also create our own interactable objects

Getting started

Sign into Unity
Create your scene and delete the main camera

Add Plane or Terrain of your choice for your character to wallk on.

Add the Player to your scene.
Assets -> SteamVR -> InteractionSystem -> Core -> Prefabs -> Player

You may want to add teleportation so that your character can move around.
See the teleportation lesson.
Prefabs

Now let's check out the built-in objects we can work with.

Open up Assets -> SteamVR -> InteractionSystem -> Samples -> Prefabs
Letâ€™s start with a ThrowableBall.

Grab one and put it in your scene next to the player.

Now try it out!
Use the trigger to grab the ball. You can change hands, hold it with both hands, throw it, drop it, pick it up, etc...

Now add the ThrowableCube and ModalThrowable, and try them out.
All three of these objects are programmed with animations for your controller's hands.
Adding your own objects

Now, let's create our own interactable objects.

Add a 3D Object of your choice (cube, sphere, capsule, cylinder, etc...)
Now resize it appropriately (we don't want it bigger than the player!)

Select the 3D Object you added, and in the Inspector, choose "Add Component".
Search for "Interactable", and add that script.

Now Add Component again, and search for "Throwable", and add that script.

Play your game and test it out! You should be able to pick up and throw your 3D object.

Notice that the hand animations don't change when you interact with this object. Now you can add your own models and make them interactable!
If you download a model from the unity store, you may need to add a collider to it in order to make it interactable and throwable.

Be sure to sign out of Unity when you're done!
Show an instructor your finished project!
Congrats! Project complete!
