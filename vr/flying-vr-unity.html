Flying

In this lesson you will:

Give the ability to fly to the Player

What You Will Make

One of the most amazing things to do in VR is make your player able to fly! To do this, we will implement an Ironman type mechanic, where pulling the triggers applies a force in the direction the controllers are pointed, to propel the player in a direction.

Getting Started

Sign into Unity

Create a new game or scene.

Create a Terrain, and get rid of the Main Camera.

Add the [CameraRig] to the game.
Assets -> SteamVR -> Prefabs -> CameraRig

Rename CameraRig to PlayerRig

Add a Rigidbody to the PlayerRig
Freeze Rotation on the Rigidbody of the PlayerRig

Add a box collider to the PlayerRig

Add the script SteamVR_ActivateActionSetOnLoad to the PlayerRig
Set the action set variable to /actions/default
Binding UI Actions

We need to create a new Vector1 action called Fly.

Open the SteamVR Input Window
Window -> SteamVR Input

Create a new action called Fly that is a Vector1.

Save and generate the actions

Open the Binding UI

Set the pull action for the trigger to fly.
Scripting

Add a new script called Player to the PlayerRig

Inside this script, we'll clamp the velocity of the PlayerRig, so we don't go too fast. This will help with motion sickness. We will also write code so that the player can't fall through the floor.

Inside the Player script, create two variables.

Inside the Start() function, connect the rigidbody variable to the Rigidbody of the PlayerRig

Create a private void FixedUpdate() function

Inside the function we need two if statements.

Save the script, then set maxVelocity in the Inspector on PlayerRig.
15 is a resonable maxVelocity.

Now open up the PlayerRig, and select both Controllers

Click Add Component and add a new script called Jetpack to both hands, then open it

Call the namespace using Valve.VR

We need to define some variables to start with.

Create the following variables.

Now inside the Start() function, we need to get the player's rigidbody

We need a private void FixedUpdate() function

Inside this function we will write the code to make you fly!
Set a float thrust to equal the input from the fly action.
Now we will use that value to apply a force to the Player's Rigidbody.

Save the script, and briefly run the game to check for errors.

Once the errors are handled, open the Controller and set the variables for Jetpack on both hands.
Set fly to the action you made
Choose an action set
Set fly multiplier to -15. This will allow us to move away from the direction we point in.

Test out your flying mechanic!

Be sure to sign out of Unity when you're done!
Show an instructor your finished project!
Congrats! Project complete!
