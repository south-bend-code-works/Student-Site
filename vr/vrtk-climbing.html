VRTK Climbing

In this lesson you will:

  1. Learn to make objects in your game climbable.
  2. Use the Setup Interactable Object menu.

What You Will Make

A climbing game!

Getting Started

Set up a new VRTK project using the steps in setup.

Create an Empty Game Object and name it PlayArea.

Add VRTK_BodyPhysics and VRTK_PlayerClimb scripts to the PlayArea.

Add some cubes in a stack to climb. You could also add a building.

Select the Objects you want to make climbable.
Hold ctrl and click multiple objects to select more than one.

Click Window->VRTK->Setup Interactable Object.

Select Climbable as the Grab Attach Mechanic.

Deselect Disable on Idle, Add Rigidbody and Add Haptics.
You can add haptic feedback if you want, by leaving Add Haptics selected.
Haptic feedback is sound or vibration that lets the user know when they touch or use an object. This is very helpful feedback in games.

Click Setup Selected Object(s). This will add all the appropriate scripts to the objects you selected.
Close the Setup Selected Objects Window.

Add VRTK_InteractTouch and VRTK_InteractGrab scripts to Left and Right Controllers.

If you haven't saved your changes, save them now.

You can change the Grab Button on the Left and Right Controllers, by default it is the grip press, but you may want to make it the trigger.
The grip press is the small buttons on both sides of the controller handles.

If you adjust the Grab Precognition to a small value like 0.1, youâ€™ll be able to grab onto objects slightly before your controller touches them.

On the climbable objects, deselect Stay Grabbed on Teleport on the Interactable Object Script.

On the VRTK_Interactable Object script, toggle Hold Button to Grab if you want to have to hold the button down to continue climbing the object. Leave it unchecked if you want to just click the button to grab on.

You can also turn on and adjust haptic feedback within the Setup Interactable Object interface, or on the VRTK_Interact Haptics script.

Play your game!
Show an instructor your finished project!
Congrats! Project complete!
