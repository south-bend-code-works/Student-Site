Firing Objects From the Controller

In this lesson you will:

Give the player the ability to fire objects from their hands

What You Will Make

In some games, we may want the ability to fire objects from the controller. We will need a couple of things to make this happen: a player, an object to fire, a UI action, and a couple of scripts to connect things.

Getting Started

Sign into Unity

Create a new game, or open the game you'd like to add this feature to.

Create a Terrain, and get rid of the Main Camera.

Add the [CameraRig] to the game.
Assets -> SteamVR -> Prefabs -> CameraRig

Rename CameraRig to PlayerRig
Fire Action UI

Open the SteamVR Input Window (Window -> SteamVR Input)

Create a boolean action called fire.

Save and generate the inputs, then open the binding UI

Change the Trigger pull action to fire. You can also use a different controller action here. See Binding UI Actions for more help.

Click "Replace Default Bindings" to save this change.
Making the Ammo

Choose a 3D object that you want to make into your ammo. I'll be using a 3D Sphere.

Create one of these, then style it as you wish.

Add a Rigidbody component to your ammo.

Once you are satisfied with your ammo, create a Prefabs folder inside Assets (if one does not exist already).

Drag the Ammo Object into the Prefabs folder to create a prefab. Then delete the original object.

Now create an empty GameObject and call it GameManager.

Deleting the Ammo

Right now our ammo hangs around forever, it;s never deleted! We don't want this to happen.

There are a variety of ways we may want to delete our ammo. For now, we'll just delete it when it falls off the edge of the terrain.

Add a new script to the Ammo prefab. You'll need to open the prefab to do this.
Call this script "AmmoControl"

Inside the Update function, write an if statement that checks the position of the ammo.
transform.position.y<0

If the position is less than 0, the object has fallen off the edge, and we can destroy it.
Destroy(gameObject);

Optionally, you can also destroy the ammo after a certain amount of time.

To do this, use Destroy(gameObject, amount of time as a float)
Destroy(gameObject, 3,0f);

Save your game and test it out!

Be sure to sign out of Unity when you're done!
