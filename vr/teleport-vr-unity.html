Teleportation

In this lesson you will:
  1. Import SteamVR into a project
  2. Add a player
  3. Add the ability to teleport around a map

What You Will Make

There are two types of teleporting you can add. The character can teleport around a space, using a teleport area, and from one spot to another, using teleport points. You can also use both in the same scene.

Teleport Area

Sign into Unity
Create a Unity project. It should have only directional light, and the SteamVR plugin should be imported into it.

Create a plane and size it how you wish.
Hierarchy -> Create -> 3D Object -> Plane

Name the plane "Floor"

Duplicate the Floor
Right click on Floor -> Duplicate

Rename this plane "Teleport Plane"
Move Teleport Plane up slightly, give it a y transform value of 0.05

Add a component to the Teleport Plane.
Search for "Teleport", and select Teleport Area

Uncheck the Mesh Renderer for Teleport Plane - this will make the plane invisible.
Teleport Points

Create a plane and size it as you wish
Hierarchy -> Create -> 3D Object -> Plane

Name the plane "Floor"

Add teleport points around your scene.
Assets -> SteamVR -> InteractionSystem -> Teleport -> Prefabs -> TeleportPoint
Both Methods

Now add the Teleporting Prefab to your Scene
Assets -> SteamVR -> InteractionSystem -> Teleport -> Prefabs -> Teleporting

Add the Player to your scene
Assets -> SteamVR -> InteractionSystem -> Core -> Prefabs -> Player

Now you can test out your game! Press play.
Use the touchpads to teleport around the plane.

Be sure to sign out of Unity when you're done!
Notes:
  • You can place a plane just above a terrain and use the plane as a teleport area.
  • The default size of a terrain is 100x the size of a plane. So to make an appropriately sized plane, change the x and z scale in the transform
  • You cannot use a terrain as a teleport area.
