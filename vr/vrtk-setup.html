VRTK Setup

In this lesson you will:
  1. Create a new Unity Project
  2. Download and setup the Virtual Reality Toolkit (VRTK) package

What You Will Make

A working VR project with VRTK!

Getting Started

Go online and download VRTK Version 3.3.0.
VRTK 3.3.0 Download

Download the SteamVR 1.2.3 Plugin. VRTK 3.3 does NOT support SteamVR 2.
SteamVR 1.2.3 Download

Click Add, and open the VRTK 3.3.0 folder as a Unity Project from Unity Hub.
Use Unity 2019.2.
If you get an error about a Non-Matching Editor Installation, continue.

Import the SteamVR1.2.3 Package (Assets -> Import Package -> Custom Package -> find package)

Click Import

Click "I Made A Backup. Go Ahead!"

Click "Accept All"
Testing your game using SteamVR.

Open SteamVR from your Desktop.
Be sure that your headset is plugged into the box and that the sensors and controllers are on.

The SteamVR window should show that all components are detected and active.

Press Play on your game.

If you haven't saved your changes, save them now.
Recalibrating the Room

Open the SteamVR application
Click on the hamburger menu

Select "Room Setup" from the menu, and follow the directions.
Setting up the VRTK SDK.

Delete the Main Camera Game Object

Create an Empty Game Object and name it VRTK_SDK.

Add VRTK_SDK Manager script to the VRTK_SDK object.

Create an Empty Game Object inside VRTK_SDK and name it SteamVR_SDK.

Add VRTK_SDK Setup script to the SteamVR_SDK object.

In the Quick Select Dropdown Select SteamVR.

Uncheck Auto Populate.

On the VRTK_SDK Manager Script on VRTK_SDK, click “Auto Populate”.
The SteamVR SDK Setup should appear in the list.

From the SteamVR Assets folder -> Prefabs, drag a [CameraRig] and a [SteamVR] Object onto the SteamVR_SDK object.
Setting up the Controllers.

Create two new empty game objects and name them LeftController and RightController.

Select the VRTK_SDK Manager and drag LeftController and RightController to the respective locations under Script Aliases.

Add the following scripts to Left and Right Controllers:
  • VRTK_Controller Events
  • VRTK_Pointer
  • VRTK_Straight Pointer Render (or Bezier Pointer Renderer)

Drag the straight pointer renderer into the Pointer Renderer box on the VRTK_Pointer script.
You will need to do this for both controllers.

On the SteamVR_SDK, click the Populate Now button to connect the Camera and controllers properly.

Test to ensure that both pointers work.
At this point, both pointers should appear when you touch the touchpads.
Show an instructor your finished project!
Congrats! Project complete!
