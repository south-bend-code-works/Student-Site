VRTK Object Interactions

In this lesson you will:

  1. Learn about the different ways to interact with objects.
  2. Play in an example scene.
Getting Started

Open an existing VRTK game.

Open the Sample Scene called InteractableObjects.
File -> Open Scene
VRTK -> Examples -> [005 - Interactions] InteractableObjects

Look around this scene to see what all you can do with different objects.
There are 38 different objects to explore. They are detailed below.
Basic Interaction Station

Near Touch Cube
Reacts when the controller is near and when the controller is touching it.

Grab Cube
Can be grabbed at a fixed joint point by the controller (same place every time)

Precision Grab Cube
Can be grabbed at any point with the controller.

Whirlygig
Spins when it is used (trigger), whether is is grabbed or not.

FlyingSaucerA
Can only be grabbed with the Left Controller.

FlyingSaucerB
Can only be grabbed with the Right Controller.
Grab Mechanic Types Station
Grab mechanics are how the user picks up objects.

Lamp Custom Joint
This collection of objects uses custom joints.

Lever Animated Grab
Runs through an animation when the object is grabbed and moved away from the origin point.

Cube Fixed Joint Grab
This cube uses a fixed joint to connect to the controller, this joint will break if the object hits another object.

Cube Spring Joint Grab
This cube uses a spring joint to attach to the controller.

Cube Child Controller Grab
This cube becomes a child of the controller when grabbed.

Cube Track Object Grab
This cube tracks the controller via Unity Physics. It will drop if the controller gets too far away.

Cube Fixed Joint Toggle Grab
This cube is grabbed on a press of the grip, and dropped on a press of the grip.

Move Object Grab
This cube follows the motion of the controller, but is constrained within a space.
Snap Handles Station
Snap handles allow the player to pick up an object in the same orientation every time.

Dummy Gun One and Two
Can only be used if held, and each press of the trigger fires. These have snap handles and will be held properly no matter how they are picked up.

Lightsaber
The first press of the trigger activates the lightsaber, and the second press deactivates it.

Lighter
The lighter has two different snap handles, one for the right controller and one for the left controller.

Fire Extinguisher
The fire extinguisher has two interactable objects, one for the base and one for the hose.
Two Hand Grab Station
These objects can be interacted with using both controllers.

Cube Uniform Resize
This cube will uniformly resize when grabbed with a second controller.
For a project using this resizable cube, check out the Builder's Playground

Cube Non Uniform Resize
This cube will resize in the direction the second controller is moving when grabbed.

Oar
The oar can be grabbed with a secondary controller which will change the direction it is pointing in.

Long Gun
The long gun can be grabbed with a secondary controller, which will control the direction the gun is pointing in.
Controllables Station
Physics Based: These use rigidbodies and Unity physics to govern interactions.

Physics Pushers
Can simulate buttons and receive external force from controllers and other objects with rigidbodies.

Physics Sliders
Can simulate objects sliding on a specific axis. Can be used to create drawers.

Physics Rotators
Can be used to simulate an object rotating around an axis using a joint, like a lever, chest hinge, dial or door hinge.
Controllables Station
Artificial Based: These simulate physics, but do not require rigidbodies.

Artificial Pushers
Uses no physics, instead listens for a collision to active, for example, a button.

Artificial Sliders
Great for attaching to other objects, as they don’t require rigidbodies. Can be used for drawers, sliders, etc...

Artificial Rotators
Can be used for doors, dials, chest hinges, levers, etc… reduces complexity by not requiring rigidbodies and physics.
Pointer Interaction Station
These objects can be interacted with using the pointer.

Whirlygig With Use Action
This whirlygig spinner will activate when the pointer is touching it.

Whirlygig With Use Action Toggle
Pointing the pointer at this whirlygig will activate it. Pointing again will deactivate it.

Toggle Pointer Interaction Lever
This lever toggles whether the pointer can be used to grab/use the objects from a distance.

Toggle Grab To Tip Lever
This lever toggles whether objects will grab to the tip of the pointer or to the controller.

Whirlygig
This whirlygig can be interacted with and grabbed by using the pointer.
What now?

To replicate any of these object settings in your project, note what scripts and settings are used in the example scene, and replicate those inside your own scene.
