Walking and Jumping with the Trackpad

In this lesson you will:
  1. Edit vive's default keybindings
  2. Allow a player to move using the vive controller's trackpad
  3. Allow a player to jump using the vive controller's trackpad

What You Will Make

A project that allows the player to move with the controller's trackpad

Getting started

Sign into Unity
Create your scene and delete the main camera

Add a plane or terrain, shape and style it as you choose.

This time we need to add the Camera Rig instead of the Player.
Assets -> SteamVR -> Prefabs -> [Camera Rig]

We need to keep the CameraRig from rotating so our players don't get motion sick.
In the inspector, add a RigidBody Component.

Within the RigidBody section of the inspector, select "Freeze Rotation" for X, Y, and Z. (You may need to click on the triangle to expand the Constraints section.)

Add a Capsule Collider to the CameraRig.
Set the radius to 0.1

Now our CameraRig is ready to go!
Binding UI Actions

For this project, we will need to bind two new UI actions:
MovementAxis, a Vector2
Jump, a boolean

Open the SteamVR Input Window from Window -> SteamVR Input

Click the + to add another Action
Name this action "MovementAxis"

Choose "vector2" as the type, then choosed "suggested" from the 'Required:' menu

Click + again to add another action
  1. Name the Action "Jump"
  2. Set the type to boolean
  3. Set the Required to suggested

Click "Save and Generate"

Now click "Open binding UI", and you'll see this open in your browser:

Select "edit for the current binding"

We're going to bind the Movement Axis to the 'Trackpad' action of the trackpad.
Click the + on the Trackpad

Select trackpad, and delete the "Use as Button" box for trackpad.

Edit the "Use as Trackpad box"
Select "Position"

Choose movementaxis

Next, we'll bind "Jump" to clicking the trackpad.
Choose the "Click" Option under Trackpad

Choose "Jump"

Great work! We've connected these actions to the trackpad input.

Click "Replace Default Binding"
You'll then see this window:

Click Save
