VRTK Builder's Playground

In this lesson you will:

  1. Scale objects with the controllers.
  2. Make a playground to build in.

What You Will Make

A building playground!

Getting Started

Set up a VRTK Project and enable some type of teleportation.
Use height adjust teleport if you want to stand on top of the structures you create.
Making Objects Scale with the Controllers

Create Cube, and resize it to about .25/.25/.25.
Color the cube by making a material of the color you want and dragging it onto the cube.

Add a Rigidbody to the cube.

Add VRTK_Interactable Object Script to the cube.
Check Is Grabbable.

Add VRTK_Axis Scale Grab Action Script to the cube.

Set the Ungrab Distance as you see fit (.25 works well).

Check Uniform Scaling for uniform scaling.

Add VRTK_Fixed Joint Grab Attach Script to the cube.

Check precision grab.

On the Interactable Object Script, drag the Fixed Joint Grab Attach Script into the Grab Attach Mechanic Script box, and drag Axis Scale Grab Script into the Secondary Grab Action Script box.
Make a Clone Drop Zone.

Make a cube and resize it to be flatter (like a table).

Rename this cube DropZone.

Check out the docs for Snap Drop Zones here.

Get a SnapDropZone prefab from Assets -> VRTK -> Prefabs -> SnapDropZone and place it as a child of the cube you just made.

Resize the sphere collider a little larger.

Move the SnapDropZone up above the DropZone cube.
Change the y location value.

Check Clone New On Unsnap.

Drag your resizable cube into the Highlight Object Prefab box.

Now play the game and try dropping your resizable cube on the drop zone.

Build away!

Try adding multiple types of resizable objects, and multiple drop zones to clone them in.
Each Cloning Drop Zone will only take one object.
Show an instructor your finished project!
Congrats! Project complete!
