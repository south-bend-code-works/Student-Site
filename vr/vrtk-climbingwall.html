VRTK Climbing Wall

In this lesson you will:

  1. Use the Setup Interactable Objects menu.
  2. Build a new project.

What You Will Make

A climbing wall game.

Getting Started

Set up a new VRTK project using the steps in setup.

Add whatever type of locomotion you'd like to use.
Teleportation is a good option.

Start by making a wall out of a cube, resizing it to be a tall wall.

Make a small sphere, make it a different color, climbable, and then duplicate and sprinkle them across the cube, making them protrude from the surface of the cube.
Tip: Ctrl+D will duplicate an item.

Tip: Store all the climbing holds in an empty game object to keep your game organized.

Add VRTK_BodyPhysics to the PlayArea so that the player can fall from the wall.

To add hand costumes instead of the default controllers, Go to Assets -> VRTK -> Prefabs -> AvatarHands -> BasicHands, and add VRTK_BasicHand prefab as a child of LeftController and RightController.

Adjust the Digital Finger settings on the BasicHands so that Thumb, Index, and Three Finger Axis are all controlled by either the trigger or grip, whichever you are using for climbing.

If you haven't saved your changes, save them now.

Test out your game!

Add a skybox and scenery to make your game more realistic.

Other ideas:
  • Add multiple walls
  • Make hand holds of different colors to create different routes.
  • Create a whole world to climb around.
  • Import a city asset and make the buildings climbable.
Show an instructor your finished project!
Congrats! Project complete!
