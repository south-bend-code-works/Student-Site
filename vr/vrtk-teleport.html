VRTK Teleportation

In this lesson you will:

  1. Learn to use VRTK Teleportation
  2. Enable gravity in your VR game!

What You Will Make

A working VR project with VRTK!

Getting Started

Open your existing VRTK game.

Add a plane or cube for you to stand on in the game.
Size and style this however you like.

Create an Empty Game Object and name it PlayArea.

Add the script VRTK_Basic Teleport to the PlayArea object.
Feel free to adjust the Blink To Color.

Test the game. Now you can teleport!
Multiple ways to teleport.

Use VRTK_Dash Teleport or VRTK_HeightAdjustTeleport for other teleporting options.
Dash Teleport fades the screen briefly when teleporting.
Height Adjust Teleport allows for changing heights when teleporting.

To enable gravity, add the VRTK_BodyPhysics Script to the PlayArea.

Try out different forms of teleportation.

If you haven't saved your changes, save them now.

To add a teleportation point, go to Assets -> VRTK -> Prefabs -> DestinationPoint. Drag a DestinationPoint onto the scene.

Add some cubes and other objects to teleport onto!

Once you've enabled VRTK_HeightAdjustTeleport and VRTK_BodyPhysics, teleport onto an object, move to the edge, and then move your head off the edge to fall.
Be careful if you get motion sick!
Take a break if you start to feel sick.
Show an instructor your finished project!
Congrats! Project complete!
