Archery Game

In this lesson you will:
  1. Add a teleport plane
  2. Allow the player to pick up and shoot a bow and arrow
  3. Add targets for the player to shoot at

What You Will Make

Let’s put together what we’ve learned so far, and introduce one new object! We will build an archery game, using teleportation, interactable objects, and the built in bow and arrow asset.


Before starting this project, you should have completed the following lessons:

  1. Adding VR to a Unity Game
  2. Teleportation
  3. Interacting with Objects
Getting started

Sign into Unity
Create your scene and delete the main camera

Let's download some assets to use for our scene.

Go to the Asset Store and search for "Nature Starter Kit."
Select, download, and import Nature Starter Kit 2.

you may get an error message about obsolete APIs. Click "I made a backup, Go Ahead! "

Download and import Unity's Standard Assets as well.

Add a terrain, shape and style it as you choose.

Add the required components for teleportation. You could use only teleportation points, just add a teleport area, or use both!

Add your player character

Test your game to make sure everything works and looks good.
Archery

Let's add the archery component.

First add a 3D cube to serve as a table for the bow to sit on.
Add this near the player, and make it about half the height of the player.

Now we'll make the longbow itself.
Add an empty game object, and place it on top of the cube that is our table.

Name the object "Longbow".

Add the following components:
  1. Interactable Script
  2. Item Package Spawn Script
  3. Sphere Collider

Now on the Item Package Spawn Script, check "Require Grab Action", "Show Trigger Hint", and "Take Back Item".

Now we'll give the Longbow an Item Package.
From Assets -> SteamVR -> InteractionSystem -> Longbow -> Prefabs, drag the LongBowItemPackage into the Item Package box on the Item Package Spawn Script.

Finally, drag a Longbow Preview from the Longbow Prefabs folder into the Longbow Game Object, so that the Preview is a child of the Longbow game object.

Let's add some targets to shoot at.

In the Longbow Prefabs folder, there is a prefab called Archery Weeble. Drag a few of these into your scene and then test out shooting arrows at them!

What if we want our arrows to be on fire?

Let's check out the torch prefab!

Drag a Torch prefab into your scene and rotate it until it's upright.

Test your game! Now if you hold an arrow into the flame, it will catch on fire. Try shooting flaming arrows at the weebles.
More things to shoot at

Let's add a stack of blocks to shoot at.

Create a 3D Cube away from the player character. This will act as a table for our blocks.

Now make smaller 3D Cubes that are Interactable and Throwable.
Tip: Make one, then duplicate it!
Stack the cubes in a creative pattern on the table.

don't forget that you can organize this in an empty game object.

Now you can add more targets, torches, and interactables to build out your game!

Be sure to sign out of Unity when you're done!
Show an instructor your finished project!
Congrats! Project complete!
